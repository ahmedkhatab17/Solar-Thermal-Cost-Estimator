# Solar Thermal Cost Estimator

A lightweight static calculator for estimating the materials and costs of a solar thermal storage system. The single-page site is built with plain HTML, CSS, and JavaScript.

## What is included

- `index.html`: interactive pricing calculator for tank size, mirror count, and project phases
- responsive UI with Arabic labels and local/import cost breakdown
- real-time cost, energy, and payback estimates

## How to use

1. Open `index.html` in any browser.
2. Adjust the tank size and number of mirrors.
3. Review the updated cost summary and thermal system breakdown.

## Deployment

This project is ready to deploy as a GitHub Pages site from the repository root.
